Source StreamExtScanSource ( void  )

Get a handle to the extended scanning stream source There is one extended scanning stream source that provides the extended advert data for all scanners. It is created either when this function is called or if extended advert data is received for an enabled scanner. It is automatically closed when there are no longer any scanners enabled, calling SourceClose() on this source will fail. This source is message based and each advert has a header of type DM_ULP_EXT_SCAN_FILTERED_ADV_REPORT_IND_T. A pointer to the next advert's header can be acquired with SourceMapHeader() and a pointer to the advert data via SourceMap(). This source does not support being connected to sinks via StreamConnect(). When this source is full new adverts coming from the air destined for this stream are dropped. SourceSizeHeader() will return non zero if there are any messages to read from the source. SourceSize() and SourceBoundary() are not a reliable way to determine if the source is empty as they will return zero if there are messages in the source with zero length data.

Returns
The source handle for the global extended scanning stream.
Source StreamPeriodicScanSource ( uint16  sync_handle)

Get a handle to a periodic scanning stream source for a sync handle Multiple periodic scanning sources can exist simultaneously, one for each active sync handle. Multiple calls to this function for a given sync handle will return the same stream. This function should only be called after obtaining the sync_handle from a DM_ULP_PERIODIC_SCAN_SYNC_TO_TRAIN_CFM prim. If called with an invalid sync handle, one that is not currently synced, this function will return zero. This source is message based and each advert has a header of type DM_ULP_PERIODIC_SCAN_SYNC_ADV_REPORT_IND_T. A pointer to the next advert's header can be acquired with SourceMapHeader() and a pointer to the advert data via SourceMap(). This source is closed automatically when a periodic sync is terminated via the DM_ULP_PERIODIC_SYNC_TERMINATE prim, or if the sync is lost. Calling SourceClose() on this source will fail. When this source is full new adverts coming from the air destined for this stream are dropped. SourceSizeHeader() will return non zero if there are any messages to read from the source. SourceSize() and SourceBoundary() are not a reliable way to determine if the source is empty as they will return zero if there are messages in the source with zero length data.

Returns
The source handle for the periodic scanning stream associated with the sync handle.
