Control the various aspects of the audio CODEC.

file IIR Filter coefficients are configurable on some Bluecore variants. This trap is used to program set of 16-bit-coefficients to multiple IIR filters on ADC channels. This trap is licensed and can be used only with licensed devices. Note that the DSP operator framework must be loaded before setting the configuration parameters. Check OperatorFrameworkEnable documentation for more details. Example: if channel_mask = 0x0003, IIR Filters of ADC A and ADC B are updated with the given set of coefficients.

Parameters
channel_maskADC channel bit mask to configure IIR coefficients on multiple ADC channels. It is a bitmask with 6 valid bits, each bit representing an ADC channel.
enableEnables (TRUE) or disables (FALSE) the filter.
iir_coefficientsA pointer to structure (IIR_COEFFICIENTS) containing 16-bit IIR coefficients.
Returns
TRUE if the filter was configured/disabled as requested, else FALSE. This will return FALSE if IIR feature is not licensed. An invalid coefficient address or an invalid mask will also result in a FALSE return.
