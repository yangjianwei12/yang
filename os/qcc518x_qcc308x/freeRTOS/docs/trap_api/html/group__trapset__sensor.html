Triggers a read of the specified sensor. This function is used to perform an immediate sensor transfer. Generally used by sensor drivers to configure a devices registers. This function will block until the results are available.

Parameters
sensor_idThe Sensor ID.
tx_dataPointer to data to transmit, NULL if nothing to transmit.
tx_sizeNumber of octets to transmit.
rx_dataPointer to buffer to receive data, NULL if nothing to receive.
rx_sizeNumber of octets to receive.
Returns
TRUE if transfer was successful, FALSE if transfer failed.
BusTransferHandle SensorTransferBegin ( Sensor  sensor_id)

Begin a sequence of asynchronous transfers This function is used to being a sequence of read and/or write transfers on the bus. Reads and write are prepared by calling SensorAddWrite() and SensorAddRead(). Once the sequence has been prepared a call to SensorTransferEnd() passes the sequence to the framework to be actioned.

Parameters
sensor_idThe Sensor ID.
Returns
A handle that used in subsequent calls to SensorAddWrite()/SensorAddRead()/SensorTransferEnd()
