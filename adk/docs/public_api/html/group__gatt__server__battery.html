Detailed Description

Header file for the GATT Battery Server module.

Creates tables of marshal type descriptors for GATT server battery data types.

Component that deals with initialising the gatt_battery_server library, and deals with messages sent from this library.

There can be multiple battery servers, so this component initialises and handles each server. Also each server could have multiple battery clients connected, if supported by the application, where each client can store and retrieve its own configuration.
