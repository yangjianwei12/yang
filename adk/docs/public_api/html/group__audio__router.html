Detailed Description

The audio router provides a standard API to enable or disable audio paths.

Provides a standard API to register and unregister audio and voice observers for use in the audio router.

Data handling functions for use in the audio router.

Storage and retrieval of dynamic data for the single entity module.

Implementation of audio router functions for single signal path.

Specific implementations of the audio router behaviour can be configured by the registration of handlers with the AudioRouter_ConfigureCallbacks() function.

This implementation can then call into the audio router to connect and disconnect sources as required.
