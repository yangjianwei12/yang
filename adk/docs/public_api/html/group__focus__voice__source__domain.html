bool Focus_GetVoiceSourceInContextArray ( ui_providers_t  provider,
voice_source_t *  voice_source,
const unsigned *  contexts,
const unsigned  num_contexts 
)

#include <adk/src/domains/focus/focus_voice_source.h>

As Focus_GetVoiceSourceForContext but limits the voice sources processed to be only those in one of the requested contexts. The voice_source will be set to the highest priority source in one of the requested contexts.

Parameters
provider- a UI Provider
voice_source- a pointer to the focussed voice_source_t handle. If the value passed in this parameter matches a voice_source symbol, other than voice_source_none, then this voice source will be excluded from the results
contexts- a pointer to the array of requested contexts
num_contexts- The number of contexts requested
Returns
a bool indicating whether or not a voice source was returned in the voice_source parameter
