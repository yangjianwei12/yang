uint8* fastPair_GetOwnerAccountKey ( void  )

#include <adk/src/domains/bt/fast_pair/manager/fast_pair.h>

Get the "owner" account key.

The "owner" account is the first Fast pair user account to be associated with a device, it should be considered special and should not be overwritten

The Google finder feature uses this account.

Returns
Pointer to The owner account key if one exists, else NULL. the calling function is responsible for freeing the memory at the returned address

Message Handler.

Handler for all messages to fast pair Module

This function is called to handle any messages sent to the fast pairing module. If a message is processed then the function returns TRUE.

Note
Some connection library messages can be sent directly as the request is able to specify a destination for the response.
Parameters
taskTask to which this message was sent
idIdentifier of the message
messageThe message content (if any)
Returns
None

This function is the main message handler for the fast pair module.
