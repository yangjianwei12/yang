◆ timeout

uint32 timeout

Scan will automatically stop after this timeout period (ms)

The timeout has a minimum of #LE_BROADCAST_SELF_SCAN_TIMEOUT_MINIMUM_MS, and a maximum of #LE_BROADCAST_SELF_SCAN_TIMEOUT_MAXIMUM_MS.

If the timeout is set to 0 then the default timeout of #LE_BROADCAST_SELF_SCAN_TIMEOUT_DEFAULT_MS is used.

If the timeout is outside the above values then the start scan request will fail with the error lebmss_bad_parameters.
