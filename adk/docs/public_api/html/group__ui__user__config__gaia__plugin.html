#include <adk/src/domains/ui/ui_user_config_gaia_plugin/ui_user_config_gaia_plugin.h>

Initialise the UI User Config GAIA Plug-in.

This function registers the GAIA command/notification handler of the End-User UI configuration feature. The Application that uses this Feature must call this function before calling any other functions of the Feature.

Returns
TRUE indicating success

◆ UiUserConfigGaiaPlugin_SetNumberOfTouchpads()

void UiUserConfigGaiaPlugin_SetNumberOfTouchpads ( uint8  num_touchpads)

#include <adk/src/domains/ui/ui_user_config_gaia_plugin/ui_user_config_gaia_plugin.h>

Register a Composite Gesture ID to Logical Input mapping.

This API is expected to be called by the Application to inform the GAIA plug-in of the number of touchpads available to the end-user. Setting 1 means the device has a single originating touchpad. Setting 2 could be a device in an Earbud pair or possibly a Headset application, where there are discrete Left and Right touchpads.

Parameters
num_touchpads- The number of touchpads
