#include <adk/src/domains/bt/profiles/le_pbp/pbp_client_source.h>

Configure broadcast streaming using PBP. Clients should handle PBP_CLIENT_MSG_ID_SRC_CONFIG_COMPLETE to check whether the configuration has successfully completed or not.

Parameters
handleProfile handle on which the streaming has to be configured.
presentation_delayencryption status for the broadcast streaming.
num_subgroupBroadcast code that set for the broadcast source role
subgroup_infocontains stream configuration parameters such as number of BIS, metadata, etc.
source_name_lenLength of Broadcast Source name.
source_nameName of Broadcast.Source.
bcast_typePublic broadcast type. Either SQ or HQ Public Broadcast.
encryptionTRUE if broadcast source is encrypted
