Detailed Description

Map of user PS Keys.

Add a PS Key here if it is going to be used. If there is possibility that a PS Key would need to be populated during deployment then it should also be added to subsys7_psflash.htf.

enum ps_read_only_key_t

#include <adk/src/domains/common/ps_key_map.h>

Application read-only keys, provide storage for critical or sensitive application data.

Total of 10 keys, these can only be accessed through PsFullRetrieve(..) using the PS Key indexes PSKEY_APPLICATION_READONLY0 - PSKEY_APPLICATION_READONLY9, and not through the apps subsytem PS key indexing used by PsRetrieve(..)
