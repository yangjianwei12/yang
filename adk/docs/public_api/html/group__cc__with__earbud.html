void CcWithEarbuds_HandsetPairing ( void  )

#include <adk/src/domains/case_comms/cc_with_earbuds/cc_with_earbuds.h>

Initiate handset pairing on Earbuds by sending command over case comms.

Note
Earbuds must be in the case and the case lid open for the handset pairing command to be accepted and them to enter handset pairing mode. A handset pairing command is sent to both left and right earbuds, only the primary Earbud will perform handset pairing, the seconadry will forward the command to the primary. Duplicate command is handled gracefully by the earbuds and avoids the case having to identify which is currently the primary earbud.

The CcWithEarbuds_HandsetPairingResponseRx() callback will be called by the case channel component for each response from the left and right earbuds indicating accept/reject of the handset pairing command.

Transmit loopback message(s) to an Earbud.

Parameters
destCasecomms device address.
dataPointer to buffer holding data to loopback.
lenLength of data in bytes.
iterationsNumber of loopback messages to send.
Note
Only the left or right earbud addresses are valid destinations.

Transmission of the loopback message(s) is marked as a timestamped event and when the same number of loopback messages are received from the earbud another timestamp is marked. The difference between these two timestamps is written to the log.
