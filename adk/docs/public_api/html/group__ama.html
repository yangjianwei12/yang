Detailed Description

Interfaces defination of Amazon AVS interfaces.

Protected API to only be used within the AMA component.

Provides TWS support in the accessory domain.

Definition of the APIs to manage AMA device feature states.

Definition of the APIs to manage the AMA speech configuration.

AMA setup tracker APIs.

Implementation for AMA On-Head/In-Ear Detection for Accessories.

Defintion of the APIs for handling AMA events.

Extended initialisation of AMA.

File consists of function decalration for Amazon Voice Service's EQ handling.

File consists of function to get/set data for ama.

Configuration for audio functionality for Amazon Voice Service.

Configuration related definitions for AMA capabilities support.

Implementation for AMA BLE.

File consists of function decalration for Amazon Voice Service's battery handling.

File consists of function decalration for Amazon Voice Service's audio specific interface.

File consists of function decalration for Amazon Voice Service's ANC handling.

Interface for handling ama user events.
