◆ GattServerGap_UseCompleteLocalName()

void GattServerGap_UseCompleteLocalName ( bool  complete)

#include <adk/src/domains/bt/gatt/servers/gatt_server_gap/gatt_server_gap.h>

Tells the GAP server to use the complete local name, or if it can be shortened.

The GAP server handles the LE advertising of local name which can be shortened depending on the space in the advertising data. This function can ensure the complete local name is always supplied in the advertising data.

Parameters
completeSet to TRUE so the complete local name is always used. Otherwise can be shortened.
