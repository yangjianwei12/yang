file  va_audio_test.h
 Interface for voice assistant audio testing functions. These override the VA protocols and are not meant to be used to drive VA feature (they are audio specific test APIs). They should not be used while an assistant is connected/used (they will override the audio control from the VA protocol and vice versa).
 

Detailed Description

These files contain APIs for testing common ADK functionality.

The functions can be used from the python based tool, pydbg using the format apps1.fw.call.test_function_name(parameters if any)

The functions should not be called from other parts of the framework or application.
