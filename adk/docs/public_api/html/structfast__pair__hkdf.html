#include <fast_pair_hkdf.h>

Detailed Description

Structure containing parameters used by FastPair_HkdfSha256

Parameters
[in]saltThe optional salt value (a non-secret random value); if not provided (salt == NULL), it is set internally to a string of SHA256_DIGEST_SIZE zeros.
[in]salt_lenThe length of the salt value. (Ignored if salt == NULL.)
[in]ikmInput keying material.
[in]ikm_lenThe length of the input keying material.
[in]infoThe optional context and application specific information. If info == NULL or a zero-length string, it is ignored.
[in]info_lenThe length of the optional context and application specific information. (Ignored if info == NULL.)
[out]okmWhere the HKDF is to be stored.
[in]okm_lenThe length of the buffer to hold okm. okm_len must be <= 255 * SHA256_DIGEST_SIZE
