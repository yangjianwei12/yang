Detailed Description

Header file for USB Mass Storage class driver.

FAT16 support for USB mass storage class.

Header file for USB Mass Storage common code.

Class specific definitions for USB Mass Storage.

USB Mass Storage class implements a read-only storage presenting arbitrary content.

The class emulates a read-only FAT16 filesystem. Long names are not supported so all file names must be 11 capital characters (8 + 3). Reserved sectors are hard-coded in the code and can not be changed.

Application provides 3 files with content for the following regions:

  1. Root Directory Region: contains names and attributes of the data files.
  2. FAT Region: FAT16 data.
  3. Data Region: content of the data files. These files shall be placed onto a read-only filesystem on the flash.

UsbMsc_PrepareConfig() can be called to initialize configuration data with the file indices and sizes.
