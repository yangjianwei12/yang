◆ LeAdvertisingManager_ConfigureAdvertisingOnNotifyDataChange()

bool LeAdvertisingManager_ConfigureAdvertisingOnNotifyDataChange ( le_adv_config_notify_t  config)

#include <adk/src/domains/bt/le_advertising_manager/legacy/le_advertising_manager_legacy.h>

Public API to configure LE advertising behaviour when LeAdvertisingManager_NotifyDataChange() API is called.

Parameters
[in]configArgument of type le_adv_config_notify_t.
Note
For LE advertising to stop and restart when LeAdvertisingManager_NotifyDataChange() API is called, "le_adv_config_notify_restart_advertising" should be passed as the parameter. For LE advertising to continue without stopping and restarting when LeAdvertisingManager_NotifyDataChange() API is called, "le_adv_config_notify_keep_advertising" should be passed as the parameter. When configured with the parameter "le_adv_config_notify_keep_advertising", LE advertising can continue using old data for a period of time until LE advertising packet is updated with the new data.
Returns
TRUE for a succesful call to API for notification, FALSE otherwise.

Handler for connection library messages not sent directly.

This function is called to handle any connection library messages sent to the application that the advertising module is interested in. If a message is processed then the function returns TRUE.
