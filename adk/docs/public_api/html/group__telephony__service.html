Detailed Description

The Telephony service provides a high level API for performing Telephony actions on a voice source, through the use of a sources telephony control interface.

Telephony service private.

Telephony service call control actions.

The Telephony service uses Audio Audio domain.

◆ TelephonyService_ConfigureCallerIdNotifications()

void TelephonyService_ConfigureCallerIdNotifications ( bool  enable_caller_id)

#include <adk/src/services/telephony/telephony_service.h>

Enable or disable Caller ID notifcations.

If Caller ID notifications are enabled in the Telephony Service, when a Voice Source becomes connected, the Telephony Service shall register to receive Caller ID notifications for that source.

Parameters
enable_caller_id- If caller ID is supported for a Voice Source, setting this TRUE means that Caller Id notifications shall be received.
