◆ appSdpFindTwsVersion()

bool appSdpFindTwsVersion ( const uint8 *  ptr,
const uint8 *  end,
uint16 *  tws_version 
)

#include <adk/src/domains/bt/sdp/sdp.h>

Extract the TWS version from SDP records supplied.

Used to extract the TWS version from a full SDP record, delineated by pointers to start (ptr) and end of the SDP.

Parameters
ptrPointer to the start of the SDP records
endPointer to the end of the SDP records
tws_versionAddress of a value to take the TWS version. This cannot be NULL.
Returns
TRUE if a TWS version was found in the SDP records, FALSE otherwise.

◆ sdp_GetGaiaDebugPeerLinkServiceRecord()

const uint8* sdp_GetGaiaDebugPeerLinkServiceRecord ( uint16 *  length)

#include <adk/src/domains/bt/sdp/sdp.h>

Accessor function to get the SDP record for the GAIA Debug Peer Link service.

This function returns the pointer to the service record for the GAIA Debug Peer Link service.

The GAIA Debug (also known as the 'Wireless Debug' over BT) uses a link between the Primary and the Secondary for Pydbg Remote Debug, which sends Pydbg commands to the Secondary and receives responses from it.

Parameters
[out]LengthPointer to value to take the record length (in bytes).
Returns
Pointer to the service record.
