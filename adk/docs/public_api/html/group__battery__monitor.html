◆ appBatteryRegister()

bool appBatteryRegister ( batteryRegistrationForm *  form)

#include <adk/src/domains/power/battery_monitor/battery_monitor.h>

Register to receive battery change notifications.

Note
The first notification after registering will only be sent when sufficient battery readings have been taken after power on to ensure that the notification represents a stable value.
Parameters
formThe client's registration form.
Returns
TRUE on successful registration, otherwise FALSE.
