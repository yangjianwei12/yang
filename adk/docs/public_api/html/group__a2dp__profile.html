◆ A2dpProfile_SetPreferredSampleRate()

bool A2dpProfile_SetPreferredSampleRate ( avInstanceTaskData *  av_instance,
uint32  rate 
)

#include <adk/src/domains/bt/profiles/a2dp_profile/a2dp_profile.h>

Set preferred sample rate for the specified AV instance.

During media channel establishment, A2DP will attempt to negotiate the preferred sample rate given here, if supported by the selected codec and both sides of the link. Currently only the aptX Adaptive source encoder supports this feature.

For codecs that do support the feature, A2DP will never negotiate a sample rate better than the expressed preference (it acts as a hard upper limit). This may result in that codec being rejected in favour of another, if there is no common supported rate that is equal to or lower than the preference. Higher/better rates can always fall back to lower rates if required, though.

If a media channel is already established, then calling this function may result in an AVDTP reconfigure command being issued, renegotiating the codec parameters to take account of the new preference. In this case, the function returns TRUE. If the preference made no difference, FALSE is returned.

Note
Renegotiation may suspend/resume the media channel, as necessary.
Parameters
av_instanceThe instance for which the preferred sample rate applies.
ratePreferred sample rate to negotiate, if possible. A value of 0 represents no preference (the default), which will negotiate the best common rate available between source & sink.
Returns
TRUE if the change in preference resulted in a reconfiguration of the media channel (which will now be in progress), FALSE otherwise.

#include <adk/src/domains/bt/profiles/a2dp_profile/a2dp_profile.h>

Message Handler.

This function is the main message handler for an A2DP instance, every message is handled in it's own seperate handler function. The switch statement is broken into seperate blocks to reduce code size, if execution reaches the end of the function then it is assumed that the message is unhandled.

Parameters
theInstThe instance data for the AV for this A2DP connection
idMessage identifier. For internal messages, see av_internal_messages
[in]messageMessage content, potentially NULL.
