Detailed Description

Header file for Bluetooth Local Address component.

Local address functions that should only be used, carefully, by other domain components.

◆ LocalAddr_OverrideBleGeneration()

local_address_context_t LocalAddr_OverrideBleGeneration ( Task  task,
local_addr_host_gen_t  host,
local_addr_controller_gen_t  controller 
)

#include <adk/src/domains/bt/local_addr/local_addr_protected.h>

Override the settings for Local Address.

The function replaces current settings (if any) with those supplied. A context is returned, to be used when the override is no longer required.

Note
Use of this function is not recommended. Earlier system settings will be replaced which could affect operation.
Parameters
taskThe Task requesting the override.
hostHost settings to apply. See LocalAddr_ConfigureBleGeneration()
controllerController settings to apply. See LocalAddr_ConfigureBleGeneration()
Returns
The old context. This should be passed to LocalAddr_ReleaseOverride() when the override is no longer needed.
