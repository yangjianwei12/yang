◆ MirrorProfile_IsVoiceSourceSupported()

bool MirrorProfile_IsVoiceSourceSupported ( voice_source_t  source)

Check if the given voice source can be mirrored to the peer.

The mirror profile does not support mirroring of SCO links or eSCO links with using the HV3 packet format.

If the handset has connected to the primary with one of the unsupported hfp SCO link types the voice source should not be mirrored to the secondary.

Parameters
sourceThe voice source to check.
Returns
TRUE if it can be mirrored; FALSE otherwise.
