reboot_action_default_state 

The component initiated reboot recommends the application/client to boot as usual Power On would boot the system. The component initiated reboot recommends the application/client to move to a default state which is well defined by the application/clients

reboot_action_t SystemReboot_GetAction ( void  )

#include <adk/src/domains/power/system_reboot/system_reboot.h>

Returns the action to be taken post reboot. When app uses this function to move to a well defined state by application, the reboot action should be reset for the next boot to be as reboot_action_default_state. To achieve this the app can call SystemReboot_ResetAction.

reboot action
