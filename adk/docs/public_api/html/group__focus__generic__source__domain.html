◆ Focus_ConfigureGenericSourceOverride()

void Focus_ConfigureGenericSourceOverride ( focus_get_generic_source_override_t const *  focus_generic_source)

#include <adk/src/domains/focus/focus_generic_source.h>

Configure override function pointers to use for retrieving the focussed generic source (currently audio only)

Clients can register the override functions that can return the focussed generic source in some scenarios that client is interested in. Override functions to be executed first to check for focussed generic source, if no active generic source is found, then logic to find focussed generic source would fall-back on functions registered through Focus_ConfigureAudioSource().

Parameters
Astructure containing the function pointers implementing the focus interface for retrieving the focussed generic source (currently audio source only) for the specified client configuration.
