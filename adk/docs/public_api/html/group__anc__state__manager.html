Detailed Description

State manager for Active Noise Cancellation (ANC).

Private header for State manager for Active Noise Cancellation (ANC).

Handles session data corresponding to ANC domain.

Quiet mode header for Adative Active Noise Cancellation (ANC).

Responsibilities: Handles state transitions between init, enable and disable states. The ANC audio domain is used by Audio Curation.

Responsibilities: Handles and persists session data corresponding to ANC domain.
