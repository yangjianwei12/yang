◆ marshal_rtime_t

typedef rtime_t marshal_rtime_t

#include <adk/src/domains/common/marshal_common.h>

A special microsecond resolution timestamp type.

When this type is marshalled, the timestamp is automatically converted from local clock domain to BT clock domain on marshalling, and from BT clock domain to local clock domain on unmarshalling. The client code therefore can transmit local clock timestamps to the peer and the timestamp received by the peer will represent same time on the peer device's local clock.
