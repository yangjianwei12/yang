Detailed Description

This is the interface to HCI TAP Stream that logs the BT HCI Interface traffic out off the chip for debugging purpose.

!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!! WARNING: calling HciTapStream_ConnectToCDCDevice or hciTapStream_ConfigureUart functions in the code by default is a security risk as the HCI Interface provides the LINK Keys.

Therefore it MUST NEVER be called in the code by DEFAULT in a final product. !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
