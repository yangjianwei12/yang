Detailed Description

Source file converts System Events to corresponding Audio Prompt UI Events by table look-up, using a configuration table passed in by the Application. It then plays these Prompts when required using the Kymera audio framework Aux path.

Message sent to clear the last prompt information (used to ignore repeated prompts)

Message sent later when a prompt is played. Until this message is delivered repeat prompts will not be played
