Detailed Description

Common types used by the Event Bus API's.

Subscribe API for the Event Bus The Event Bus component is provided for the centralized distribution of events throughout the system, decoupling event publishers from their subscribers.

Publish API for the Event Bus The Event Bus component is provided for the centralized distribution of events throughout the system, decoupling event publishers from their subscribers.

Debug logging level stuff for the Event Bus.
