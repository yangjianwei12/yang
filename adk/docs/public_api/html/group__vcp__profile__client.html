void VcpProfileClient_RegisterNotification ( void  )

#include <adk/src/domains/bt/profiles/le_vcp/client/vcp_profile_client.h>

This API is used to write the client characteristic configuration of the Volume State and volume flag characteristic on a remote device, to enable notifications with the server.

Parameters
profileHandleThe Profile handle.
notificationsEnableSet to TRUE to enable notifications on the server, FALSE to disable them.

NOTE: VCP_VOLUME_STATE_SET_NTF_CFM message will be sent to the registered application Task.
