◆ AppTouchSensorGetDormantConfigureKeyValue()

bool AppTouchSensorGetDormantConfigureKeyValue ( dormant_config_key *  key,
uint32 *  value 
)

#include <adk/src/domains/sensor/touch/touch.h>

Obtain the key/value to pass to DormantConfigure to allow the touch sensor interrupt to wake the device from dormant mode. The caller may just pass the returned values into the DormantConfigure trap, or modify them to enable other wake sources on the same key.

Parameters
[out]keyThe key to be passed to DormantConfigure
[OUT]value The value to be passed to DormantConfigure
Returns
TRUE if the touch sensor is able to wake the chip from dormant and the key/value are valid, otherwise FALSE.

◆ TouchSensor_Reset()

bool TouchSensor_Reset ( bool  hold)

#include <adk/src/domains/sensor/touch/touch.h>

Reset the touch sensor.

Called to reset the touch sensor; used at boot time to put the sensor into a known state; also when the device is taken in/out of the case or the ear, in case the user has accidentally touched the surface.

Parameters
holdIf TRUE will try to hold the sensor in reset

◆ TouchSensorClientRegister()

bool TouchSensorClientRegister ( Task  task,
uint32  size_action_table,
const touch_event_config_t *  action_table 
)

#include <adk/src/domains/sensor/touch/touch.h>

Register a UI action table and client task.

This registers a task to receive UI events. If an action_table is supplied it replaces an earlier table.

It is possible to not supply an action_table. If no call has been made to register an action table then no messages will be sent.

Parameters
taskTask to register
size_action_tableSize of action table (if any)
[in]action_tableTable converting touch events to UI events (if any)
