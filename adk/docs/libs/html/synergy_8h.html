◆ SynergyEnableLEATasks()

void SynergyEnableLEATasks ( void  )

Enable LE Audio Tasks and Message handlers based on the selected library configuration. Eg: 1. For sink based devices it enables Unicast server and Broadcast sink functionalities.

  1. For source based devices it enables Unicast client, Broadcast Source and Broadcast assistant functionalities.

This API should be called before initializing the Synergy's task scheduler using SynergyInit().

Returns
None.
Note
Application can selectively use individual tasking functionalities also based on the requirement. Eg: If application requirement is just to use Unicast server functionalities then it can invoke only SynergyEnableLEAUnicastServerTasks()
