◆ GattVocsClientAudioLocationRegisterForNotificationReq()

void GattVocsClientAudioLocationRegisterForNotificationReq ( ServiceHandle  clntHndl,
bool  notificationsEnable 
)

This API is used to write the client characteristic configuration of the Audio Location characteristic on a remote device, to enable notifications with the server. An error will be returned if the server does not support notifications.

Parameters
clntHndlThe service handle for this GATT Client Service.
notificationsEnableSet to TRUE to enable notifications on the server, FALSE to disable them.

NOTE: A GATT_VOCS_CLIENT_AUDIO_LOCATION_SET_NTF_CFM message will be sent to the registered application Task.
