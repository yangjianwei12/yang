Detailed Description

This structure contains all the Codec Configuration data collected from both the client and the Profile/Application throughout the Codec Configuration procedure. This structure can be retrieved by the Profile/Application by calling the GattAscsReadCodecConfiguration() function. The structure can be requested by the Profile/Application at any time after the ASE has completed the Codec Configuration procedure (and before the ASE is released back to an IDLE state).
