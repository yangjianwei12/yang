◆ GattGmasClientGetDeviceDataReq()

GattGmasClientDeviceData* GattGmasClientGetDeviceDataReq ( ServiceHandle  clntHndl)

This API is used to retrieve the Service device data stored by the Service library during discovery procedure.

Parameters
clntHndlThe service handle for this GATT Client Service.
Returns
GattGmasClientDeviceData : The structure containing start and end handles info. If the handles are not found or any other error happens in the process, NULL will be returned.

NOTE: This is not a message passing based API, the handles, if found, will be returned immediately by the function.
