void GattBassClientAddSourceRequest ( ServiceHandle  clntHndl,
GattBassClientAddSourceParam *  param,
bool  noResponse,
bool  longWritte 
)

This API is used to write the Broadcast Audio Scan Control point characteristic in order to execute the Add Source operation.

Parameters
clntHndlThe service handle for this GATT Client Service.
paramData of the Broadcast Source to add. I't responsability of the application to free the memory (if it exists) associated to the metadata_value contained in param.
noResponseTRUE if the operation has to be executed with the GATT Write Without Response procedure, FALSE otherwhise.
longWritteTRUE if GATT long write procedure to be used.

NOTE: A GATT_BASS_CLIENT_ADD_SOURCE_CFM message will be sent to the registered application Task.

◆ GattBassClientBroadcastReceiveStateRegisterForNotificationReq()

GattBassClientStatus GattBassClientBroadcastReceiveStateRegisterForNotificationReq ( ServiceHandle  clntHndl,
uint8  sourceId,
bool  allSource,
bool  notificationsEnable 
)

This API is used to write the client characteristic configuration of one or all the Broadcast Receive State characteristics on a remote device, to enable notifications with the server. An error will be returned if the server does not support notifications.

Parameters
clntHndlThe service handle for this GATT Client Service.
notificationsEnableSet to TRUE to enable notifications on the server, FALSE to disable them.
allSourceIf TRUE all the Client Characteristic Configurations will be written and source_id parameter will be ignored.
sourceIdSource id of the Broadcast Receive State characteristic whose Client Characteristic Configuration is asked to be written. This will be ignored, if allSource is TRUE.
Returns
GattBassClientStatus Result of the operation

NOTE: A GATT_BASS_CLIENT_BROADCAST_RECEIVE_STATE_SET_NTF_CFM message will be sent to the registered application Task.
