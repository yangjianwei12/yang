◆ GattCsisClientGetHandlesReq()

GattCsisClientDeviceData* GattCsisClientGetHandlesReq ( ServiceHandle  clntHndl)

This API is used to retrieve the CSIS characteristic and descriptor handles stored by the profile library during discovery procedure.

Parameters
clntHndlThe service handle for this GATT Client Service.
Returns
GattCsisClientDeviceData : The structure containing characteristic and descriptor handles info. If the handles are not found or any other error happens in the process, NULL will be returned.

NOTE: This is not a message passing based API, the handles, if found, will be returned immediately to the profile library.

void GattCsisClientInit ( AppTask  appTask,
const GattCsisClientInitParams *const  clientInitParams,
const GattCsisClientDeviceData *  deviceData 
)

Initialises the CSIS Client Library.

Initialize CSIS client library handles, It starts finding out the characteristic handles of CSIS.

Once the initialisation has been completed, GATT_CSIS_CLIENT_INIT_CFM will be received with status as enumerated as GattCsisClientStatus. '' has to be considered initialisation of library is done successfully and all the required charecteristics has been found out

NOTE:This interface need to be invoked for every new gatt connection when the client wish to use csis client library

Parameters
appTaskThe client task that will receive messages from this Service.
clientInitParamsConfiguration data for client initialisation. The memory allocated for GattCsisClientInitParams must be freed once the API returns.
deviceDataCached handles/data from previous connection with Peer Device OR NULL if this is the first time this peer device has connected.

NOTE: GATT_CSIS_CLIENT_INIT_CFM will be received with a GattCsisClientStatus status code..
