device_t UsbDongleSm_GetCurrentSink ( void  )

Allows other components to know which is the active sink device.

If connected, this will be the currently connected sink, otherwise, it will be the sink device we were previously connected / intend to connect to.

Returns
device_t - The current sink device, if any.
