◆ UsbDongle_A2dpAptxAdHqSampleRateGet()

uint32 UsbDongle_A2dpAptxAdHqSampleRateGet ( void  )

Return the saved aptX Adaptive HQ mode sample rate.

Note
This may not be equal to the actual current (live) sample rate, if it was not possible to negotiate the requested sample rate in the current mode (or if the current codec is not aptX Adaptive).
Returns
[out] The saved aptX Adaptive HQ sample rate, in Hz (or 0 if no saved rate i.e. override disabled).
